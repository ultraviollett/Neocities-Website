Definitions

HTML — aka "HyperText Markup Language", code used to create webpages. We can move and style this text by using CSS.

CSS — aka "Cascading Style Sheets", code meant to add style and structure to a webpage. Without CSS, webpages would different sizes of text and some inputs. There wouldn't be any colour and very limited formatting.

Ao3 Workskin — Ao3 allows you to add CSS to your work with 'workskins'. You create a workskin on Ao3, then add it to your fic, and the HTML in your fic will be able to reference the CSS in the workskin to use.

I have links for different resources below that explain different things easier, but these are the basics.

How do I use the workskin workshop?

  1. Decide what you want to add to your fic.
  2. Go to the CSS generator and generate what you will use, then add it to a workskin. There is a more indepth tutorial on creating workskins here.
  3. Generate the HTML you need with the generators. Add it to your fic (when you're in HTML mode, not Rich Text mode).
  4. Good job! It should all be working. If you notice anything off, or need any help, send me a message.
